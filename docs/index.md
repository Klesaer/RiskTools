# RiskTools Documentation

RiskTools is a documentation and knowledge hub for pricing, risk management, and balance sheet analytics. The repository is structured to support both technical implementation guidance and risk-management content for topics such as the pricing engine, the Kamarura Risk Manager (KRM), IRRBB, liquidity, FTP, balance sheet management, and FRTB.

## Project overview

This documentation set is intended to provide a practical reference for:

- understanding the pricing engine and its outputs,
- documenting the operating model and governance of KRM,
- explaining key risk topics in a structured and auditable way,
- collecting examples, implementation notes, and supporting material.

## Documentation scope

The current documentation structure covers:

- Pricing engine methodology and workflows
- KRM operating model and risk governance
- Risk topics including IRRBB, liquidity, FTP, balance sheet management, and FRTB
- Examples and API reference material for future expansion

## Intended audience

This documentation is designed for:

- quantitative analysts and model developers,
- risk managers and control teams,
- treasury and ALM specialists,
- technical teams responsible for implementation and integration.

## Next steps

The documentation will continue to evolve with:

- deeper technical content for the pricing engine,
- formal KRM process documentation,
- detailed risk-topic chapters with methodology and controls,
- example notebooks, scripts, and implementation guidance.
